##Roman Numeral

Build a function that will take any integer (1 to 1000) and return its Roman Numeral counterpart.

```
romanize(1000) #=> M
romanize(900)  #=> CM
romanize(700)  #=> DCC
romanize(500)  #=> D
romanize(450)  #=> CDL
romanize(225)  #=> CCXXV
romanize(75)   #=> LXXV
romanize(37)   #=> XXXVII
romanize(18)   #=> XVIII
romanize(9)    #=> IX
romanize(4)    #=> IV
romanize(2)    #=> II
romanize(1)    #=> I
```
